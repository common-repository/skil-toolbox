Register custom post type and include MailChimp Class for the Skil theme

== Description ==

**The Skil Toolbox plugin is meant to be used only with the Skil WordPress theme.**

This plugin register custom post type and include MailChimp Class that are needed in the [Skil theme](http://wordpress.org/themes/skil/)

== Installation ==

1. Go to Plugins > Add New and search for Skil Toolbox. Install and activate it.
2. That's it. No configuration needed.
